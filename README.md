## 📖 Overview

AlgoVision is a cutting-edge web application designed to visualize complex algorithms beautifully and interactively. Built with a modern tech stack, this application allows users to experience algorithms in real-time with an elegant UI and robust performance. Whether you are a student learning data structures, a developer preparing for technical interviews, or an educator teaching algorithm concepts, AlgoVision provides the tools you need to understand and communicate algorithmic concepts effectively.

The platform transforms abstract algorithmic processes into dynamic, step-by-step visual representations, bridging the gap between theoretical knowledge and intuitive comprehension. Every animation, color choice, and interactive element has been thoughtfully implemented to enhance memory retention and conceptual understanding, making it an invaluable resource for learners at all levels.

## ✨ Features

### Core Visualization Features

| Feature | Description |
|---------|-------------|
| **Interactive Step-by-Step Visualization** | Observe algorithms one operation at a time with precise control over execution speed |
| **Variable Speed Control** | Transition seamlessly between rapid overviews and methodical examinations |
| **Custom Data Input** | Provide your own arrays, graphs, or tree structures for personalized learning |
| **Performance Analytics** | View detailed metrics including comparisons, swaps, and space complexity analysis |
| **Real-time Graphs** | Visualize operation distribution with dynamic performance charts |

### Algorithm Categories

| Category | Algorithms Supported |
|----------|---------------------|

## 🚀 Getting Started

### Prerequisites

Before installing AlgoVision, ensure that you have the following prerequisites installed on your machine:

| Requirement | Version | Notes |
|-------------|---------|-------|
| **Node.js** | ≥ 18.0.0 | LTS version recommended |
| **npm** | ≥ 9.0.0 | Comes with Node.js |
| **Git** | ≥ 2.30.0 | For version control |

You can verify your Node.js and npm versions by running the following commands in your terminal:

```bash
node --version
npm --version
git --version
```

### Installation

Follow these steps to set up AlgoVision on your local machine:

#### Step 1: Clone the Repository

```bash
git clone https://github.com/dustin04x/algovision.git
cd algovision
```

#### Step 2: Install Dependencies

```bash
npm install
```

This command will install all the required dependencies defined in `package.json`. The installation typically completes within 2-3 minutes depending on your internet connection speed.

#### Step 3: Start the Development Server

```bash
npm run dev
```

The development server will start automatically. Open your browser and navigate to:

```
http://localhost:5173
```

You should see the AlgoVision application running with hot module replacement enabled.

#### Step 4: Build for Production

```bash
npm run build
```

This command creates an optimized production build in the `dist` directory. The build process performs minification, tree-shaking, and other optimizations.

#### Step 5: Preview Production Build

```bash
npm run preview
```

Preview the production build locally to verify correct behavior before deployment.

## 🧪 Testing

AlgoVision maintains a comprehensive testing strategy that ensures reliability and prevents regressions.

### Testing Philosophy

Our testing approach follows the testing pyramid principle, with a strong foundation of unit tests supplemented by integration and end-to-end tests. This strategy provides comprehensive coverage while maintaining fast test execution times.

### Running Tests

#### Unit and Integration Tests (Vitest)

```bash
# Run all tests
npm run test

# Run tests in watch mode during development
npm run test:watch

# Run tests with coverage report
npm run test:coverage

# Run tests for specific files
npm run test -- algorithm.test.ts
```

#### End-to-End Tests (Playwright)

```bash
# Install Playwright browsers (first time only)
npx playwright install

# Run all e2e tests
npx playwright test

# Run specific test file
npx playwright test example.spec.ts

# Run tests with UI mode
npx playwright test --ui

# Debug tests
npx playwright test --debug
```

#### Running All Checks

```bash
# Run linting, type checking, and tests
npm run check
```

## 📊 Performance

AlgoVision is optimized for smooth visualization even with large datasets:

| Metric | Target | Achieved |
|--------|--------|----------|
| First Contentful Paint | < 1.5s | ✅ |
| Largest Contentful Paint | < 2.5s | ✅ |
| Time to Interactive | < 3.0s | ✅ |
| Bundle Size (gzipped) | < 200KB | ✅ |
| Lighthouse Performance Score | > 90 | ✅ |

### Optimization Strategies

- **Code Splitting**: Components are loaded on-demand to reduce initial bundle size
- **Memoization**: Strategic use of `useMemo` and `useCallback` prevents unnecessary re-renders
- **Virtualization**: Large data sets are rendered efficiently using windowing techniques
- **Web Workers**: Computationally intensive algorithm steps run in background threads

## 🧩 API Reference

AlgoVision provides a programmatic API for advanced users and developers who want to integrate algorithm visualization into their own projects.

### Algorithm Runner

```typescript
import { AlgorithmRunner } from '@algovision/core';

const runner = new AlgorithmRunner({
  algorithm: 'quick-sort',
  data: [64, 34, 25, 12, 22, 11, 90],
  speed: 1000, // milliseconds per step
  onStep: (state) => console.log(state),
  onComplete: (result) => console.log(result),
});

// Start the visualization
runner.start();

// Pause execution
runner.pause();

// Resume execution
runner.resume();

// Reset to initial state
runner.reset();

// Stop and cleanup
runner.destroy();
```
